GNU Emacs 一起從零開始到精通 01 安裝與熟悉操作

Posted on Dec 4 2017

What is Emacs

一種上古時代的編輯器
至今仍然有許多人用著它
它與另一個編輯器Vim齊名
其最大的特性是本身就是個Lisp環境
除了核心部份其餘都是用EmacsLisp開發而成
因此提供了強大的拓展性與自定義性
號稱什麼都能在上面做
另一個特色是需要運用大量快捷鍵
不過這個特色並不是絕對
也有很多高手搭配插件僅使用指令
來達到不亞於快捷鍵的速度
因此快捷鍵的記憶並不是最重要的

How to install Emacs

首先建議使用Linux或是其他UnixLike作業系統
因為Windows有點不方便調試
不過做到還是不會很難

我的作業系統是Kubuntu 17.04
如果不是用Apt的就自行轉換囉

即可安裝最新版本的Emacs
我的版本是25.1.1

接著執行

如果要使用終端機版的 Emacs

但是終端機版的快捷鍵在某些複雜的組合下會有點問題
不過網路上都找得到很好的解決方案

你就會發現一個醜醜的界面
反正以後會讓它變好看
就先丟著吧

Key-Maps

Emacs 最難入門的就是快捷鍵了
可以先點擊Emacs開始頁面上的”EmacsTutorial”來熟悉基本操作

如果是終端版的則可以按下Alt x之後輸入 help-with-tutorial來開啟教學

簡單的幾點就是

M = alt
C = ctrl

快捷鍵表中用’-‘連接的兩個鍵要一起按
否則依序按就可以了

M-x 開啟可輸入指令的Buffer 按tab可以自動補全指令
C-x C-f 開檔或是開新檔 (open file) 按tab可以自動補全路徑和檔案
C-x C-s 存檔 (save)
C-g 關閉Minibuffer (遇到快捷鍵按錯可以用)
C-s 搜尋 (search)
C-x 0 關閉目前視窗的顯示
C-x o 切換視窗
C-k 關閉Buffer (kill)
C-x C-c 退出Emacs
C-y 貼上 (yank)
C-w 剪下
M-w 複製

差不多這幾個比較常用到

用M-x輸入指令後它也會提醒你這個指令綁到什麼快捷鍵上
用C-h 加上 快捷鍵 就能知道有什麼快捷鍵用它開頭

基本上多用幾次就記起來了
找不到再Google就行了
